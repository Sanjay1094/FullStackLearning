Basic HTML

HTML stands for Hyper Text Markup Language.

It is a markup language used to create web pages. HTML is used to describe the structure of a web page and its content. HTML consists of a series of elements that define different parts of a web page such as headings, paragraphs, links, images, and more.Refer us

Characteristics of HTML

  1. Html is the simplest language that can be easily understood and modified.
  2. It gives the flexibility to design the web pages so that a structured display can be possible for all listed documents on the web pages.
  3. Formatting tags could be used for effective presentations in the web portal, and it could possibly because of HTML.
  4. Multiple links can be added to the web page to redirect to other pages using listed links easily.
  5. The most important part of HTML is, it can be displayed in Macintosh, Windows, and Linux and support all environments. Html is platform-independent.

Applications

Browsers like Chrome, Firefox, Safari all use HTML to serve the web contents for better display. Different mobile browsers like Opera, Firefox focus, Microsoft edge, dolphin, and puffin all use HTML for better presentation and visibility of internet content in mobile. Different smart devices are embedded with HTML functions for better browsing and navigation during their operation. HTML supports a primary authentication channel mechanism to any of the web pages to stop unwanted traffic. HTML accommodates the large content but gives the same visibility for small screen devices and large screen devices.

Advantages

  1. HTML is a platform Independent.
  2. It is widely and globally accepted.
  3. Every browser supports HTML.
  4. It is easy to learn, use and modify.

  6. It uses templates which makes the website design easier.
  7. It supports a wide range of colors, formats, and layouts.
